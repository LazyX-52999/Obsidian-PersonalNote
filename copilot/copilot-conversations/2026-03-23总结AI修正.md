## 多态参数详细解释

### 什么是多态参数？
多态参数是指**方法的形参类型声明为父类类型，但实际调用时可以传入子类对象作为实参**。这是Java多态性的一个重要应用。

### 核心概念
```java
// 父类
class Animal {
    public void makeSound() {
        System.out.println("动物发出声音");
    }
}

// 子类
class Dog extends Animal {
    @Override
    public void makeSound() {
        System.out.println("汪汪汪");
    }
}

class Cat extends Animal {
    @Override
    public void makeSound() {
        System.out.println("喵喵喵");
    }
}

// 使用多态参数的方法
public class Test {
    // 形参类型是Animal（父类）
    public static void animalSound(Animal animal) {
        animal.makeSound();  // 运行时动态绑定到实际对象的makeSound方法
    }
    
    public static void main(String[] args) {
        Animal animal = new Animal();
        Dog dog = new Dog();
        Cat cat = new Cat();
        
        // 实参可以是Animal或其子类对象
        animalSound(animal);  // 输出：动物发出声音
        animalSound(dog);     // 输出：汪汪汪  ← 多态参数
        animalSound(cat);     // 输出：喵喵喵  ← 多态参数
    }
}
```

### 多态参数的特点

1. **向上转型（Upcasting）**：
   ```java
   Animal animal = new Dog();  // 子类对象赋值给父类引用
   ```

2. **运行时多态**：实际调用哪个方法由运行时对象的实际类型（运行类型）决定

3. **提高代码灵活性**：
   ```java
   // 一个方法可以处理多种类型的对象
   public void processAnimals(List<Animal> animals) {
       for (Animal animal : animals) {
           animal.makeSound();  // 可以是Dog、Cat或其他Animal子类
       }
   }
   ```

### 实际应用场景

1. **集合处理**：
   ```java
   List<Animal> zoo = new ArrayList<>();
   zoo.add(new Dog());
   zoo.add(new Cat());
   zoo.add(new Lion());
   
   // 一个方法处理所有动物
   feedAllAnimals(zoo);
   ```

2. **回调方法**：
   ```java
   public void registerListener(EventListener listener) {
       // listener可以是任何EventListener的子类
   }
   ```

3. **设计模式中的应用**：
   - 策略模式
   - 观察者模式
   - 模板方法模式

### 注意事项

1. **只能调用父类声明的方法**：通过父类引用只能调用父类中定义的方法
2. **方法重写（Override）**：子类需要重写父类方法才能实现多态效果
3. **编译时类型 vs 运行时类型**：
   - 编译时类型：声明时的类型（Animal）
   - 运行时类型：实际对象的类型（Dog、Cat等）
