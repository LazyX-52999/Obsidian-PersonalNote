#### JavaSE
今日学习了 ==类变量== 、 ==类方法== 、 ==main方法==

类变量：
	又叫：静态变量/静态属性
	类变量最大的特点是：类变量会被本类中的所有对象实例==共享==
	类变量是随着类的加载而创建的（即：类在加载的时候就生成了类变量）。因此类变量可以直接访问（不用再创建对象来访问），只要类加载了就可以使用类变量
类变量的访问方法：
	类名.类变量名
	对象名.类变量名
	==注：实例变量 不可以通过 	类名.类变量名 来访问==
类方法：
	又叫：静态方法
	==静态方法 只能访问 静态成员==
	==普通方法 可以访问 静态成员 + 非静态成员==
	（成员 = 属性 + 方法）
	静态方法不能使用 ==this== 和 ==super==
类方法的访问方法：
	类名.类方法名
	对象名.类方法名
	==注：非静态方法 不可以通过 类名.类方法名 来调用
	==
==注：类变量和类方法都要遵守访问修饰符的访问权限的规则==

main方法：
	main方法是虚拟机调用的
	main方法可以访问本类中的静态方法和静态变量，
	但不能访问本类中的非静态方法和非静态变量（可以通过创建对象来访问）
	
``` java
	public static void main(String[] args) {}
	IDEA简写：psvm
	
	public：java虚拟机需要调用类的main方法（java虚拟机不在同一个类中），所以访问权限是public
	static：java虚拟机在执行main方法的时候不用创建对象
	void：无返回类型
	main(String[] args)：该方法接收的是字符串类型的数组，该数组中保存执行java命令时传递给所运行的类的参数
```
