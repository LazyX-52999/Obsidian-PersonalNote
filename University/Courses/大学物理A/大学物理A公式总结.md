### 1.2质点运动的描述
位置矢量：** $\overrightarrow{r}$ **
质点的运动方程：**$$ \begin{array}{l} \vec {r} = x (t) \vec {i} + y (t) \vec {j} + z (t) \vec {k} \\ o r  \left\{ \begin{array}{l} x = x (t) \\ y = y (t) \\ z = z (t) \end{array} \right. \\ \end{array} $$**
质点的轨迹方程：对质点的运动方程消 t 
*例: 由运动方程求轨道方程
已知质点的运动方程为: $\vec{r} = a\cos \pi t\vec{i} + b\sin \pi t\vec{j}$ 求质点的轨道方程。
解: $x = a \cos \pi t, y = b \sin \pi t$
消去 $t \rightarrow \frac{{x}^{2}}{{a}^{2}} + \frac{{y}^{2}}{{b}^{2}} = 1$*

位移(质点位置矢量的增量)： **$\Delta \overrightarrow{r} = \overrightarrow{r}^{\prime } - \overrightarrow{r}$**
位移的大小(即位移矢量的模长)：
**$$ |\Delta \overrightarrow{r}| = \sqrt{(\Delta x)^2 + (\Delta y)^2 + (\Delta z)^2} $$**
路程(质点运动曲径的长度 标量)：**$\Delta{s}$**

平均速度：**$\overline{\vec{v}} = \frac{\Delta \vec{r}}{\Delta t}$**
瞬时速度(速度)： **$\vec{v} = \lim_{\Delta t \to 0} \frac{\Delta \vec{r}}{\Delta t} = \frac{d \vec{r}}{dt}$**
瞬时速率(速率)： **$v = |\vec{v}| = \left|\frac{d\vec{r}}{dt}\right| = \frac{ds}{dt}$**

在直角坐标系中：
瞬时速度： **$\overrightarrow{v} = \frac{d \vec{r}}{dt} = \frac{dx}{dt}\vec{i} + \frac{dy}{dt}\vec{j} + \frac{dz}{dt}\vec{k} = {v}_{x}\overrightarrow{i} + {v}_{y}\overrightarrow{j} + {v}_{z}\overrightarrow{k}$**
瞬时速率： **$v = |\vec{v}| = \sqrt{v_{x}^{2} + v_{y}^{2} + v_{z}^{2}}$**
速度的方向(用方向余弦表示为)：**$\cos\alpha = \frac{v_x}{v}, \cos\beta = \frac{v_y}{v}, \cos\gamma = \frac{v_z}{v}$**，其中$\alpha, \beta, \gamma$分别是速度矢量与$x, y, z$轴的夹角。

平均加速度：**$\vec{\bar{a}} = \frac{\Delta \vec{v}}{\Delta t}$**
瞬时加速度(加速度)：**$$ \vec {a} = \lim _ {\Delta t \rightarrow 0} \frac {\Delta \vec {v}}{\Delta t} = \frac {d \vec {v}}{d t} = \frac {d ^ {2} \vec {r}}{d t ^ {2}} $$**

### 1.3曲线运动

圆周运动：
切线坐标轴：**$\overrightarrow{e_{t}}$ **
法向坐标轴：**$\overrightarrow{e_{n}}$**
速度： **$\overrightarrow{v} = v \overrightarrow{e_{t}} = \frac{ds}{dt} \overrightarrow{e_{t}}$**
加速度：**$$ \overrightarrow {a} = \overrightarrow {a _ {t}} + \overrightarrow {a _ {n}} = \frac {d v}{d t} \overrightarrow {e _ {t}} + \frac {v ^ {2}}{\rho} \overrightarrow {e _ {n}} $$**
匀速圆周运动：
1. 特征:速率恒定，轨迹为圆
2. 法向加速度(向心加速度)：$\overrightarrow{a}_{n}$
   方向：$\overrightarrow{a}_{n} \perp \vec{v}$ ，指向圆心
   大小：$a_n = \frac{v^2}{R}$

变速圆周运动：
1. 特征：速度大小、方向都变，轨迹为圆
2. 法向加速度：$\overrightarrow{a}_{n}$
   方向：指向圆心
   大小：$a_n = \frac{v^2}{R}$
3. 切向加速度：$\overrightarrow{a}_{t}$
   方向：切向 $\vec{a}_t \parallel \vec{v}$
   大小：${a}_t = |\vec{a}_t| = \frac{dv}{dt}$
4. 总加速度 $\overrightarrow{a} = \overrightarrow{a_{t}} + \overrightarrow{a_{n}}$
   方向：$\vec{a}$与$\vec{a}_t$夹角$\tan \theta = \frac{a_n}{a_t}$
   大小：$a = \sqrt{a_n^2 + a_t^2} = \sqrt{\left(\frac{v^2}{R}\right)^2 + \left(\frac{dv}{dt}\right)^2}$
